This is a fast, read-oriented custom editor. It has a polished table browser, virtualized grid, per-column filters, row detail modal, BLOB/image handling, bottom pagination, file association, theme integration, export command, and web-compatible worker architecture. It is not a marketing or toy interface, but a restrained database workbench that is quiet, dense, and predictable.

This project should be a full local SQLite workbench:

- Browse database objects quickly.
- Edit table data safely.
- Manage schema objects.
- Preserve VS Code save, undo, revert, and dirty-state expectations.
- Keep the UI quiet, dense, predictable, and native to VS Code.
